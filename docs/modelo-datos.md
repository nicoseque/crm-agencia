# Modelo de Datos - CRM Agencia

## Entidades principales
- users
- leads
- clients
- vehicles
- deals
- activities

## Objetivo
Gestionar el ciclo completo de ventas desde lead hasta cierre.

## Relaciones
- Un usuario gestiona muchos leads
- Un lead puede convertirse en cliente
- Un cliente puede tener múltiples operaciones
- Cada operación tiene seguimiento
